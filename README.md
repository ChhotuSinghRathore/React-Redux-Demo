## Getting Started

### Installing
Dependencies 
  1. apollo-boost
  2. graphql
  3. react-apollo
  4. redux
  5. react-redux

How to run app
  1. Open project in VS code
  2. Open terminal(cmd+j) and execute npm install command
  3. react-native run-android
